About E-Commerce Website

Welcome to ExploreWithSarim — a smart and simple e-commerce app that showcases live products using modern frontend development techniques.

This project fetches data from a public API and dynamically renders products with category filters and add-to-cart functionality. It's built using pure HTML, CSS, and JavaScript.

The goal is to provide a clean, responsive interface while practicing real-world development skills. This is part of my portfolio and ongoing learning.

Thank you for exploring — feedback is welcome!
